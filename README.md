# temperature-mortality
Code repo for "Understanding and addressing temperature impacts on mortality". The repo contains basic scripts to replcation main statistical results in the paper. The underlying EU and Mexico mortality data are public and are provided below, with merged temperature data; the US data are restricted use and available by application to the CDC, and are not provided.
